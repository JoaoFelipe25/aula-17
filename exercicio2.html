HTML

    HTML e uma linguagem de marcação utilizada para desenvolver sites para web.
    É composto por elementos definidos por meio de tags,
    e suporta estilização e script externos como por exemplo javascript.
    
CSS

    
    O CSS  é a sigla para cascade style sheet ou folha de estilo em cascata.
    É utilizada para estilizar componentes em uma pagina HTML.  
    pode ser usada inline dentro da tag, ou fora de uma tag style ou ate msm em um arquivo externo.
